Denon AVC Plugin (DenonAVC)
===========================

! [Logo plugin] (../asset/images/DenonAVC_icon.png "Logo plugin")

Plugin para controlar el amplificador de la serie AVC de Denon a través de Jeedom.

> ** Nota **
> Este plugin fue desarrollado porque el plugin oficial (denonavr) no funcionó en mi amplificador
> Explota la interfaz web.


** Características: **

- Encender / apagar,
- Cambiar entrada,
- Controlar el volumen y activar / desactivar el silencio,

** Modelos compatibles **
- Denon AVC A11XV
- Probablemente otros modelos Denon.

Configuración de plugin
=======================

Después de descargar el plugin, active el plugin.

Configuración del equipo
========================

Se puede acceder a la configuración del equipo Denon desde el menú * Plugins > Multimedia > Amplificador AVC Denon *.

### pestaña Equipo:

- ** Denon Equipment Name **: nombre de su equipo (ex A11XV),
- ** Objeto padre **: indica el objeto padre al que pertenece equipos,
- ** Categoría **: las categorías del equipo (puede pertenecer a varias categorías),
- ** Habilitar **: para activar su equipo,
- ** Visible **: hace que su equipo sea visible en el dashboard,
- ** Dirección IP **: dirección IP del amplificador.
- ** Fuentes **: Lista de fuentes para mostrar en el mosaico

### Pestaña Pedidos

Los comandos básicos se generan automáticamente.
